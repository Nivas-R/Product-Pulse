## Overview
Product Pulse is a data-driven web application designed to analyze product performance by combining inventory data, pricing, ratings, reviews and sentiment insights. The project demonstrates how raw e-commerce data can be transformed into meaningful business intelligence using data analytics and visualization.

## Problem Statement
Online sellers often struggle to understand:
- Which products perform well or poorly
- How customer sentiment affects sales
- Inventory gaps and overstock risks
- Data scattered across multiple formats

Product Pulse addresses this by consolidating data and presenting actionable insights through a single dashboard.

## Solution
Product Pulse processes structured product data, cleans and organizes it, performs analysis, and visualizes results in a user-friendly web interface. It helps simulate how a shop owner or analyst can monitor product trends and customer feedback effectively.

## Features
- Product-wise inventory tracking
- Price, rating, and review analysis
- Sentiment-based insights from reviews
- Category-wise product listing
- Interactive data visualizations
- Clean and responsive web interface
  
## Project Workflow
1. Collect sample e-commerce product data
2. Clean and preprocess data using Excel and Python
3. Categorize products based on type
4. Perform analysis on price, stock, ratings, and sentiment
5. Visualize insights using charts and tables
6. Deploy the application using Streamlit
   
## Tech Stack
- Programming Language: Python
- Data Handling: Excel, CSV
- Libraries: Pandas, NumPy, Matplotlib, Plotly
- Web Framework: Streamlit
- Version Control: Git, GitHub
  
## Data Used
- Product details (name, category, price)
- Inventory stock levels
- Ratings and reviews
- Derived sentiment labels

> Note: The project uses sample data to demonstrate real-world analytics workflows.
